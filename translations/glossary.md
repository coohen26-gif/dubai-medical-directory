## Règles de translittération

- **Noms propres (AR)** : translittération latine commune UAE (ex: محمد → Mohammed, فاطمة → Fatima). Si le nom est déjà latin (Dr. Smith), on conserve tel quel.
- **Noms propres (RU)** : translittération ISO 9 / passeport international (ex: Сергей → Sergey, Елена → Elena). Cyrillique natif conservé pour version russe.
- **Noms propres (ZH)** : Hanyu Pinyin diacritique standard (ex: 王 → Wáng, 李 → Lǐ). Caractères chinois natifs pour version chinoise.
